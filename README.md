### Instalar yarn, en caso no se tenga:

```
npm install --global yarn
```

### Luego dentro del proyecto, instalar sus dependencias con el siguiente comando:
```
yarn install
```

### Ejecutar el proyecto con el siguiente comando:
```
yarn serve
```
